# Portfolio

A personal developer portfolio — a single-page site. Ocean teal-blue theme, dark/light mode, and GSAP-powered transitions. Resume.md file for the content/data.

## Commands

- **Dev server**: `npm run dev` (runs on http://localhost:5173)
- **Build**: `npm run build`
- **Preview production build**: `npm run preview`
- **Lint**: `npm run lint`
